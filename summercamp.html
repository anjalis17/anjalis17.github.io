Our Summer Camps

A fun introduction to robotics!

Course Overview

If you are planning to join the Vex community, sign up for our summer camp! Our course will aim to teach students about the Vex IQ pieces, common structures such as a four-bar, SnapCAD, and basic programming.

Schedule

Class Lesson Plan
Class 1 | Monday, 7/12/21 Introductions; 2021-22 VEX IQ Game & Strategy Discussion; VEX IQ Parts Lesson
Class 2 | Tuesday, 7/13/21 VEX IQ Parts Review: Jeopardy Game; Gear Ratios Lesson
Class 3 | Wednesday, 7/14/21 Structural Design Lesson; Intro to SnapCAD: Sammy
Class 4 | Thursday, 7/15/21 SnapCAD Day 2 & Application/Build Activities- Each pair CADs for the team
Class 5 | Monday, 7/19/21 Engineering Design Process & Engineering Notebook Principles; SnapCAD Wrap-Up;
Intro to Programming Fundamentals
Class 6 | Tuesday, 7/20/21 Types of Sensors & Why Sensing is Critical; Programming Fundamentals Day 2:
Variables, Functions, Control Structures, Implementing Sensing; Using Vexcode
Class 7 | Wednesday, 7/21/21 Wrap Up Programming; Begin Final Project
Class 8 | Thursday, 7/22/21 Finishing up Final Project; Final Project Show and Tell;
Opportunities/Pathways to Pursue Competitive VEX IQ
